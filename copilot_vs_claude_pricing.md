# GitHub Copilot vs Claude 料金比較ガイド

実際に使い続ける上で重要なのがコストです。
GitHub Copilot と Claude の料金体系をまとめます。

この記事では、まず一般的な料金体系を整理した上で、実際に私が GitHub Copilot をヘビーに使っている実例を紹介します。

**用語の整理**：

Anthropic が提供する Claude には複数の利用方法があります：

- **Claude.ai** - Web/アプリでの利用（Free/$17-20/$100-200の3プラン）
- **Claude Code** - CLI ツール（ターミナルで動作、VS Code拡張機能あり）。Claude.ai アカウントで無料利用可能、Pro/Max で使用枠拡大
- **Claude API** - トークンベースの従量課金 API（Developer向け）

本記事では、VS Code での開発を前提に、主に **GitHub Copilot** と **Claude Code**（VS Code 拡張機能）の料金を比較します。

## GitHub Copilot の料金体系

### 基本プラン

- Copilot Free は $0/月（月300のPremium requestsが含まれる）
- Copilot Individual は $10/月（年払いで$100/年）
- Copilot Business は $19/ユーザー/月
- Copilot Enterprise は $39/ユーザー/月

### Premium Request SKUs（追加リクエスト）

GitHub Copilot では、Claude Sonnet 4.5 などの高性能モデルを使用する場合、Premium Request SKUs を購入します。

追加リクエストの料金体系：

- 料金は約$10で、追加のプレミアムリクエスト枠を購入
- 対象モデルは Claude Sonnet 4.5、GPT-4o、その他プレミアムモデル
- 基本枠として月に一定回数のプレミアムリクエストが含まれている
- 基本枠を超えた場合、都度 SKU を購入する形式

特徴：

- 月額固定費と従量課金の組み合わせ（Freeプランは月300のPremium requests無料枠あり）
- プレミアムモデル（Claude Sonnet 4.5など）を使う場合、チャットでも補完でもPremium requestsを消費
- 標準モデルでの補完やチャット機能は基本料金内で使い放題
- プレミアムリクエストの消費は、長い会話や複雑なタスクで増加

## Claude の料金体系

Claude.ai のサブスクリプションプランと、開発者向けの Claude API があります。

### Claude.ai プラン（Claude Code を含む）

**Free プラン**：$0/月
- Web、iOS、Android、デスクトップで利用可能
- Claude Code（CLI・VS Code 拡張機能）が利用可能（使用量制限あり）
- コード生成、データ可視化、コンテンツ作成など基本機能

**Pro プラン**：$17/月（年払い）または $20/月（月払い）
- Freeの全機能に加えて、**より多くの使用枠**
- Web と ターミナルで Claude Code にアクセス
- Sonnet 4.5 と Opus 4.5 へのアクセス
- 小規模コードベース向け

**Max プラン**：$100/月（5x）または $200/月（20x）
- Pro の全機能に加えて、**さらに多くの使用枠**（5倍または20倍）
- より高い出力制限
- 会話間のメモリ機能
- 大規模コードベース向け

**重要なポイント**：
- Claude Code は**無料で使用可能**（Anthropic アカウントがあれば）
- Pro/Max プランは使用枠が増えるだけで、機能自体は変わらない
- VS Code 拡張機能も CLI も同じ Claude Code で、会話履歴を共有

### Claude API（開発者向け従量課金）

Claude API は**トークンベース**の従量課金制で、開発者がプログラムから直接利用するためのものです。

#### Claude Sonnet 4 (2025年1月時点の最新料金)

- 入力トークンは $3.00 / 1M トークン
- 出力トークンは $15.00 / 1M トークン
- キャッシュヒット（入力）は $0.30 / 1M トークン
- キャッシュ書き込み（入力）は $3.75 / 1M トークン

#### 参考: 他のモデル

- Claude Opus 4 は入力 $15.00 / 出力 $75.00 / 1M トークン
- Claude Haiku 4 は入力 $0.25 / 出力 $1.25 / 1M トークン

**特徴**：

- 完全従量課金（使った分だけ支払い）
- 月額固定費なし
- プロンプトキャッシュを活用すると、繰り返し使う部分のコストが90%削減
- 会話の長さや複雑さに応じてコストが変動

**プロンプトキャッシュとは**：

Claude API では、同じコンテキスト（例えばコードベース全体）を繰り返し送信する場合、2回目以降はキャッシュされた部分のコストが大幅に削減されます。

- 通常の入力トークン：$3.00 / 1M トークン
- キャッシュヒット（入力）：$0.30 / 1M トークン（90%削減）

例：10万トークンのコードベースを毎回送信する場合
- 1回目：$0.30（通常料金 + キャッシュ書き込み）
- 2回目以降：$0.03（キャッシュヒット）

同じコードベースで複数の会話をする場合、コストが大幅に下がります。

**注意**：Claude API は開発者向けのプログラマティックな利用を想定したもので、VS Code での日常的なコーディング支援には Claude Code（Free/Pro/Max プラン）の方が適しています。

## コスト比較シミュレーション

### ケース1: ライトユーザー（月に軽い作業のみ）

- Copilot は $0/月（Freeプランで月300 Premium requests以内なら無料）
- Claude Code (Free) は $0/月（使用量制限あり）
- Claude Code (Pro) は $17-20/月
- このケースでは Copilot Free または Claude Code (Free) が最も安い

### ケース2: ミドルユーザー（毎日開発、週に数回プレミアムモデル使用）

- Copilot は $0/月（Freeプラン）+ 月300を超えた分のSKU購入 = $0-20/月程度
- Copilot Individual は $10/月 + SKU $10-30/月 = $20-40/月
- Claude Code (Pro) は $17-20/月（使用枠内なら追加費用なし）
- 月300リクエスト以内なら Copilot Free、超える場合は Claude Code (Pro) が有利

### ケース3: ヘビーユーザー（大規模リファクタリング、長時間のコード生成）

- Copilot Free は $0/月 + SKU $80-120/月 = $80-120/月
- Copilot Individual は $10/月 + SKU $80-120/月 = $90-130/月
- Claude Code (Max) は $100-200/月（使用枠内なら追加費用なし）
- 使用量が多い場合は Claude Code (Max) の使い放題が有利

## 私の実際の使用例

ここからは、私が実際に GitHub Copilot をどう使っているか、そのコストがどうなっているかを紹介します。

### 使い方

私は VS Code + GitHub Copilot + Claude Sonnet 4.5 で、開発に関わる作業のほぼ全てを任せています。

- 開発ドキュメントの作成
- 設計の相談と整理
- コードの作成と修正
- リファクタリング
- テストコードの生成
- PR の文章作成

主に Agent モードで使用し、プレミアムモデル（Claude Sonnet 4.5）を中心に利用しています。

### 実際のコスト（2025年実績）

**年間実績**：
- 総リクエスト数：6,160リクエスト
- 年間支払額：$186.37
- 月平均：約$15.53

**月別の傾向**：
- 年始はライトに使用（月数百リクエスト程度）
- 年末に向けて使用量が増加
- 12月単月：2,508リクエスト、$88.28

**2026年1月（1月1日～7日）**：
- 現在：272リクエスト
- 支払額：$0（月300の無料枠内）

### 使用プラン

**Copilot Free プラン（$0/月）** を使用しています。

- 月300の Premium requests 無料枠が含まれる
- 無料枠を超えた分は、都度 SKU を購入
- Individual プラン（$10/月）にアップグレードしても、Premium requests の枠は変わらない
- つまり、**Free プランのままで問題ない**

### 現在の選択と今後の検証

実は Claude Code も試していますが、現状は GitHub Copilot をメインに使っています。

**今 Copilot を使っている理由**：

- すでに Copilot での作業フローが確立している
- まだ Claude Code（VS Code 拡張機能）を使い込めていない
- Agent モードに慣れている

**Claude Code について検証したいこと**：

Claude Code は VS Code 拡張機能として利用でき、Copilot と似た使い勝手を提供します。Copilot 経由で Claude Sonnet 4.5 を使うのと、Claude Code で直接 Sonnet 4.5 を使うのは、モデル自体がほぼ同じなので結果も似ている可能性があります。

大きな違いは**料金体系**です：

- **Copilot**：月300リクエストまで無料、超過分は都度SKU購入（$0.04/リクエスト）
- **Claude Code**：Free プランで使用可能、Pro プラン（$17-20/月）で使用枠拡大

私の2025年実績（年間6,160リクエスト、$186.37）を Claude Code で計算すると：
- 月平均513リクエスト → Pro プランなら $17-20/月 × 12 = $204-240/年
- Copilotの $186より少し高い

ただし、Claude Code Pro は**使い放題**なので、使用量を気にせず作業できる点が魅力です。また、12月のような2,500リクエストの月でも追加費用が発生しません。

現時点では月$15-90程度なら、慣れた Copilot で問題ないと感じていますが、使用量が増える傾向にあるため、Claude Code Pro ($17-20/月)の使い放題も魅力的です。

## 一般的な使い分け提案

私の実例を踏まえて、一般的な使い分けの指針をまとめます。

### GitHub Copilot を選ぶべき場合

#### 1. コード補完機能を重視する場合

- コード補完、インラインチャット、エディタ統合が充実
- 標準モデルでの補完は基本料金内で使い放題

#### 2. 使用量が少ない場合

- 月300リクエスト以内なら完全無料
- たまにしかプレミアムモデルを使わない場合に最適

#### 3. 都度課金が気にならない場合

- 使った分だけ支払うスタイルが好みの場合
- 月ごとの変動が大きい使い方をする場合

### Claude Code を選ぶべき場合

#### 1. 使い放題が良い場合

- Pro プラン（$17-20/月）で使用枠内なら追加費用なし
- 使用量を気にせず作業できる
- 月に300リクエスト以上使う場合はこちらが有利

#### 2. VS Code で AI アシスタントを使いたい場合

- VS Code 拡張機能でCopilot と似た使い勝手
- CLI ツールとしてターミナルからも利用可能
- GitHub/GitLab との深い統合

#### 3. コストが予測可能な方が良い場合

- 月額固定で予算管理しやすい
- ヘビーに使っても上限が分かっている
### ハイブリッド戦略

複数のツールを併用する方法もあります。

**例**：Copilot + Claude Code

- **コード補完**：Copilot の標準モデル（無料）
- **AI アシスタント**：Claude Code（Pro プラン $17-20/月）

この組み合わせなら、日常の補完は無料で使いながら、AI アシスタント機能は使い放題になります。

## まとめ

### 料金面での判断基準

各パターンに応じた選択肢は次の通りです。

- **月300リクエスト以内のライトユーザー**：Copilot Free または Claude Code (Free) が最も安い（無料）
- **月300-1000リクエスト程度のミドルユーザー**：Claude Code (Pro) $17-20/月 が有利
- **月1000リクエスト以上のヘビーユーザー**：Claude Code (Max) $100-200/月 の使い放題が有利
- **予算の予測可能性を重視する場合**：Claude Code (Pro/Max) のサブスクリプション

どちらを選んでも、AI アシスタントを効果的に使うことで開発効率は大きく向上します。
