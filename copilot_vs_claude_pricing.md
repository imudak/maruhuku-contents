## はじめに

実際に使い続ける上で重要なのがコストです。
GitHub Copilot と Claude の料金体系をまとめます。

この記事では、まず一般的な料金体系を整理した上で、実際に私が GitHub Copilot をヘビーに使っている実例を紹介します。

**用語の整理**

Anthropic が提供する Claude には複数の利用方法があります。

- **Claude.ai** - Web/アプリでの利用（Free/$17-20/$100-200の3プラン）
- **Claude Code** - **ターミナルで動作する CLI ツール**。Pro/Max プラン、または API キーで利用可能
- **Claude API** - トークンベースの従量課金 API（$3/$15 per 1M tokens）
- **VS Code での Claude 利用** - Claude API を使う拡張機能経由（Claude Code とは別物）

**重要**：VS Code で Claude を使う場合、次の2つの方法があります。

1. **Claude.ai Pro/Max サブスクリプション** - ブラウザでの Claude.ai + Claude Code CLI + 優先アクセス
2. **Claude API を直接使用** - VS Code 拡張機能経由で従量課金（月額固定費なし）

本記事では、VS Code での開発を前提に、**GitHub Copilot** と **Claude の各利用方法**の料金を比較します。

## GitHub Copilot の料金体系

### 基本プラン

- Copilot Free は $0/月（月300のPremium requestsが含まれる）
- Copilot Individual は $10/月（年払いで$100/年）
- Copilot Business は $19/ユーザー/月
- Copilot Enterprise は $39/ユーザー/月

### Premium Request SKUs（追加リクエスト）

GitHub Copilot では、Claude Sonnet 4.5 などの高性能モデルを使用する場合、Premium Request SKUs を購入します。

追加リクエストの料金体系：

- 料金は約$10で、追加のプレミアムリクエスト枠を購入
- 対象モデルは Claude Sonnet 4.5、GPT-4o、その他プレミアムモデル
- 基本枠として月に一定回数のプレミアムリクエストが含まれている
- 基本枠を超えた場合、都度 SKU を購入する形式

特徴：

- 月額固定費と従量課金の組み合わせ（Freeプランは月300のPremium requests無料枠あり）
- プレミアムモデル（Claude Sonnet 4.5など）を使う場合、チャットでも補完でもPremium requestsを消費
- 標準モデルでの補完やチャット機能は基本料金内で使い放題
- プレミアムリクエストの消費は、長い会話や複雑なタスクで増加

## Claude の料金体系

- **Opus 4.5 は利用不可**
- Web、iOS、Android、デスクトップで利用可能
- 混雑時は優先度が下がり、応答が遅くなる可能性あり

**Pro プラン**：$17/月（年払い）または $20/月（月払い）
- メッセージ数：約45回/5時間（約216回/日）
- Claude Code CLI：週40〜80時間（Sonnet 4使用可能）
- 全モデルへのアクセス：Sonnet 4.5、**Opus 4.5**、Haiku 3.5
- リセット：5時間ごと（ローリングウィンドウ方式）
- コンテキストウィンドウ：200Kトークン
- 優先アクセス：混雑時でも安定
- 小規模コードベース向け
- **VS Code では**：Claude.ai の認証を使って Claude API 経由で利用（CLI とは別）

**Max プラン**：$100/月（5x）または $200/月（20x）

**重要なポイント**：

- 2025年8月28日以降、週次制限が導入（パワーユーザー対策）

### Claude API（開発者向け従量課金）

Claude API は**トークンベース**の従量課金制で、開発者がプログラムから直接利用するためのものです。

#### Claude Sonnet 4 (2025年1月時点の最新料金)

- 入力トークンは $3.00 / 1M トークン
- 出力トークンは $15.00 / 1M トークン
- キャッシュヒット（入力）は $0.30 / 1M トークン
- キャッシュ書き込み（入力）は $3.75 / 1M トークン

#### 参考: 他のモデル

- Claude Opus 4 は入力 $15.00 / 出力 $75.00 / 1M トークン
- Claude Haiku 4 は入力 $0.25 / 出力 $1.25 / 1M トークン

**特徴**：

- 完全従量課金（使った分だけ支払い）
- 月額固定費なし
- プロンプトキャッシュを活用すると、繰り返し使う部分のコストが90%削減
- 会話の長さや複雑さに応じてコストが変動

**プロンプトキャッシュとは**：

Claude API では、同じコンテキスト（例えばコードベース全体）を繰り返し送信する場合、2回目以降はキャッシュされた部分のコストが90%削減されます。

- 通常の入力トークン：$3.00 / 1M トークン
- キャッシュヒット（入力）：$0.30 / 1M トークン（90%削減）

例：10万トークンのコードベースを毎回送信する場合

- 1回目：$0.30（通常料金 + キャッシュ書き込み）
- 2回目以降：$0.03（キャッシュヒット）

同じコードベースで複数の会話をする場合、コストが10分の1になります。

**VS Code での利用について**：

VS Code で Claude を使う場合、次の2つの選択肢があります。

1. **Claude.ai Pro/Max サブスクリプション（$17-200/月）**
   - Claude.ai Web インターフェース + Claude Code CLI + VS Code での利用が全て含まれる
   - VS Code 拡張機能は Claude.ai の認証を使って API 経由でアクセス
   - 月額固定で使用量の上限が明確（45回/5時間など）

2. **Claude API を直接使用（従量課金）**
   - VS Code 拡張機能に API キーを設定
   - 使った分だけ支払い（$3/$15 per 1M tokens）
   - Free プランユーザーでも API キーがあれば全機能が使える
   - ライトユーザーの場合、サブスクリプションより安くなる可能性が高い

**どちらを選ぶべきか**：
- 月に数百回以上使う → サブスクリプション（定額で安心）
- 月に数十回程度 → API 直接（使った分だけで安い）
- Claude Code CLI も使いたい → サブスクリプション必須

- **コード補完 + AI アシスタント（月300回まで）**：Copilot Free（無料）
- **追加の AI アシスタント**：Claude API 直接利用（使った分だけ、月$10-30程度）

**例2**：Claude.ai Pro のみ

- VS Code での開発、Claude Code CLI、Web インターフェースが全て使える
- 月額 $17-20 の定額で使用量を気にせず作業できる
- Copilot を完全に置き換える

**例3**：Copilot Free のみ（最もコスパ重視）

- 月300回のPremium requestsを有効活用
- 超過分のみ SKU 購入（約$0.04/リクエスト）
- AI アシスタント機能も無料で使える

## まとめ

### 料金面での判断基準

各パターンに応じた選択肢は次の通りです。

- **月300リクエスト以内のライトユーザー**：Copilot Free が最も安い（無料、VS Codeで AI アシスタントも使える）
  - または Claude API 直接利用（月$1-5程度、Free プランでも API キーがあれば使える）
- **月300-1000リクエスト程度のミドルユーザー**：Claude.ai Pro $17-20/月 が有利（約45回/5時間 + Claude Code CLI 週40-80時間）
  - または Claude API 直接利用（月$10-30程度、使用量に応じて変動）
- **1日100回以上使うヘビーユーザー**：Claude.ai Max $100-200/月 の使い放題が有利（Max 5x: 約225回/5時間、Max 20x: 約900回/5時間）
  - または Claude API 直接利用（月$50-150程度、プロンプトキャッシュで削減も）
- **予算の予測可能性を重視する場合**：Claude.ai Pro/Max のサブスクリプション（5時間ごとにリセット）
- **従量課金が良い場合**：Claude API 直接利用（使った分だけ、Free プランでも使える）
- **Claude Code CLI も使いたい場合**：Claude.ai Pro 以上が必須（API 認証では CLI 不可）

どちらを選んでも、AI アシスタントを効果的に使うことで開発効率は大きく向上します。
